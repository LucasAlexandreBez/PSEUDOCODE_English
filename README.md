## Declare
Used to declare variables or method references.
Examples:
- Declare and instantiate an `int` variable to store value `x`.
- Declare a method `start()` to initialize the system.

## Method / End Method
Used to define methods and mark the end of a method block.
Example:
```
Method void setup()
End Method
```

## Create
Used to initialize new object instances.
Example:
Create 2 `Animal` objects.

## Set
Used to assign a value to a variable or setter.
Example:
Set the value of `x`.

## Add
Used to add objects or values to arrays, lists, or variables.
Examples:
- Add variable `x` to ArrayList `y`.
- Add value `x` to variable `b`.

## Remove
Used to remove objects or values from arrays or variables.
Examples:
- Remove value `x` from ArrayList `y`.
- Remove the stored value from variable `a`.

## Update
Used to modify a variable or collection.
Examples:
- Update ArrayList `y` with value `x`.
- Update variable `b` with new value.

## Repeat / End Repeat
Used to define loops and repeated logic.
Examples:
- Repeat for each object `x` in list `y`.
- Repeat while `x` is less than list size.

## Call
Used to call a method on a class or object.
Example:
Call method `process(String value)`.

## Get
Used when a variable receives a value from a method or input.
Example:
Get the user input.

## Check
Used to validate or verify a value.
Example:
Check the user input using `checkUser(String x)`.

## Increment / Decrement
Used to increase or decrease numeric values.
Examples:
- Increment `x`.
- Decrement `x`.

## If / Else / End If
Decision-making structures.
Example:
```
If x equals 8
    Decrement x
Else
    Increment x
End If
```

## Switch / End Switch
Used for multi-branch decisions.
Example:
```
Switch option
    Case 1: Set x to 10
    Case 2: Set x to 20
End Switch
```

## Display
Used to print output.
Example:
Display "Success!"

## Try / Catch / End Try
Used to model error handling.
Example:
```
Try
    Call riskyOperation()
Catch error
    Display "Error occurred"
End Try
```

## For Each
Iterate through each element in a collection.
Example:
```
For each item in list
    Display item
End For
```

## Class / End Class
Object-oriented structure definition.
Example:
```
Class User
    Attribute name
    Attribute age

    Method login()
    End Method
End Class
```

## Comment
Used to annotate logic or intentions.
Example:
# Validate user input

## Import
Represents module or dependency import.
Example:
Import Math Utilities

## Break / Continue
Flow-control inside loops.
Examples:
- Break the loop
- Continue to next iteration

## Wait
Represents delay or timed pause.
Example:
Wait 5 seconds
